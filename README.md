Este repositorio contiene una serie de ejercicios diseñados para fortalecer conceptos clave mediante la implementación de soluciones estructuradas y eficientes.

---

## 🎯 Objetivos del Repositorio

El propósito de este apartado es demostrar el uso aplicado de los cuatro pilares fundamentales de la POO:

* **Abstracción:** Modelado de objetos del mundo real.
* **Encapsulamiento:** Protección de datos mediante modificadores de acceso.
* **Herencia:** Reutilización de código y jerarquías de clases.
* **Polimorfismo:** Flexibilidad mediante interfaces y sobrecarga/anulación de métodos.

---

## 📂 Contenido de los Ejercicios

Aquí encontrarás soluciones a problemas comunes como:
1.  **Gestión de Clases:** Creación de atributos y métodos.
2.  **Uso de Constructores:** Inicialización correcta de objetos.
3.  **Relaciones entre Clases:** Composición y agregación.
4.  **Manejo de Excepciones:** Robustez en el flujo de los programas.
